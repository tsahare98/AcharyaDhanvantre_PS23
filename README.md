# 📄 Document Retriever System

> A simple Python-based document retrieval system with basic AI-powered search capabilities.

---

## 🚀 Overview

This project is a lightweight document retriever built using Python. It allows users to search and retrieve relevant documents using keyword matching along with a touch of AI-based similarity for better results.

The goal of this project is to demonstrate how intelligent search can be implemented in a simple and efficient way without complex architecture.

---

## ✨ Features

* Keyword-based document search
* Basic AI-powered similarity matching
* Fast and simple execution
* Lightweight (single file implementation)
* Easy to understand and modify

---

## 🛠️ Tech Stack

* **Language:** Python
* **Core Concepts:**

  * Text processing
  * Basic Natural Language Processing (NLP)
  * Similarity matching

## ⚙️ How It Works

1. Documents are stored or loaded in the script
2. Text is processed (cleaning, tokenization, etc.)
3. User enters a search query
4. Query is compared with documents using:

   * Keyword matching
   * Basic similarity logic (AI touch)
5. Most relevant results are displayed
